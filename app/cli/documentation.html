Sweet Dreams Bakery - E-commerce System Documentation

Application Description

Sweet Dreams Bakery E-commerce System is a comprehensive order management application designed for pastry shops to efficiently handle customer orders through an online platform.

This system allows bakery staff to create, manage, and track customer orders with automatic VAT calculations and secure data storage.

System Features

Order Creation

Create new customer orders with complete client information and product selection.

View All Orders

Display all existing orders in a comprehensive list format.

Order Lookup

Find and display specific orders using order ID.

VAT Calculation

Automatic calculation of 21% VAT on all orders.

How to Use Each Functionality

  1. Enter the unique order identifier code
  2. Fill in complete customer information (name, address, email, phone)
  3. Select products from the available pastry menu
  4. Specify quantities for selected items
  5. Submit the form to calculate total with VAT
  6. Order is automatically saved to the system

This functionality displays all orders stored in the system:

  • Shows order ID, customer name, and total amount with VAT
  • Orders are displayed in a clean, readable format
  • Each order appears on a separate line for easy scanning
  • Useful for inventory management and order tracking

To view details of a specific order:

  1. Enter the exact order ID in the search field
  2. Click the search button
  3. View complete order details including customer information and products
  4. See the calculated total with VAT included

Technical Information

Frontend Technologies:
  • HTML5 for structure
  • CSS3 with Bootstrap for styling
  • JavaScript for client-side interactions
Backend Technologies:
  • PHP for server-side processing
  • Binary file storage for data persistence
  • HTTPS for secure communications
